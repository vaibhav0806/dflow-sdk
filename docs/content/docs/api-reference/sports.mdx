# Sports API

The Sports API provides filtering options for sports-related prediction markets, including available leagues and teams.

## Access

```typescript
const dflow = new DFlowClient();
const sports = dflow.sports;
```

## Methods

### getFiltersBySports

Get available sports with their leagues and teams.

```typescript
const filters = await dflow.sports.getFiltersBySports();

for (const sport of filters.sports) {
  console.log(`${sport.sport}: ${sport.leagues.join(', ')}`);
}
```

**Returns:** `SportsFilters`

```typescript
interface SportsFilters {
  sports: SportsFilter[];
}

interface SportsFilter {
  sport: string;
  leagues: string[];
  teams?: string[];
}
```

## Examples

### List All Sports and Leagues

```typescript
const filters = await dflow.sports.getFiltersBySports();

console.log('Available sports:\n');

for (const sport of filters.sports) {
  console.log(`ðŸ† ${sport.sport}`);
  console.log(`   Leagues: ${sport.leagues.join(', ')}`);
  if (sport.teams?.length) {
    console.log(`   Teams: ${sport.teams.slice(0, 5).join(', ')}...`);
  }
  console.log('');
}
```

### Find Events by Sport

```typescript
const filters = await dflow.sports.getFiltersBySports();

// Find NFL markets
const nfl = filters.sports.find(s =>
  s.leagues.some(l => l.toLowerCase().includes('nfl'))
);

if (nfl) {
  const results = await dflow.search.search({ query: 'NFL' });
  console.log(`Found ${results.events.length} NFL-related events`);
}
```

### Get Teams for a League

```typescript
const filters = await dflow.sports.getFiltersBySports();

for (const sport of filters.sports) {
  if (sport.teams && sport.teams.length > 0) {
    console.log(`${sport.sport} teams:`);
    for (const team of sport.teams) {
      console.log(`  - ${team}`);
    }
  }
}
```
